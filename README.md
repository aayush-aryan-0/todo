# Java Desktop Todo List
## Overview

A simple offline desktop task manager built using JavaFX.
The application allows users to create, update, and delete tasks with persistent storage using SQLite.

### Features

- Add new tasks

- Update existing tasks

- Delete tasks

- Task list display using a graphical interface

- Persistent task storage

### Tech Stack

- Language: Java

- GUI Framework: JavaFX

- Database: SQLite

- Build Tool: Apache Maven
